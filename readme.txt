Project Name: college library management system ( Android App ).
Aug 2014 - Oct 2014
Description:
	1) Catalog management: College library can manage books catalog and user data.
	2) Student/User login: Students, faculty and staff can create and manage their own account and access library resources.
			They can search books by name or tags with single click.

Overview:
	1) Folder 'App Source Code - Android' contains app source code.
	2) Folder 'Server Source Code - NodeJs' contains server side source code in nodejs.
